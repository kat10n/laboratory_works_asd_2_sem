# Лабораторная работа 2. Сортировка и поиск
## Галанова Екатерина Алексеевна М8О-102БВ-25

Сортировка методом пузырька и бинарный поиск по ключу.

## Структура проекта

- `struct.h` — описание структуры `Item` (поля `key`, `value`).
- `bubble_sort.c` / `bubble_sort.h` — сортировка пузырьком.
- `binary_search.c` / `binary_search.h` — рекурсивный бинарный поиск по ключу.
- `read_file.c` / `read_file.h` — чтение данных из файла:
    - `get_list_size` — получение количества элементов из первой строки файла
    - `read_file` — чтение массива элементов из файла
- `cli.c` / `cli.h` — интерфейс командной строки.
- `main.c` — точка входа, запуск CLI.
- `tests.c` — 5 тестов через `assert`.

## Формат входного файла

```
<количество элементов>
key
value
key
value
...
```

## Сборка

```
make all        - собрать основную программу (main.exe)
make tests      - собрать тесты (tests.exe)
make clean      - удалить скомпилированные файлы
```

## Запуск

```
make run        - запустить программу
make run_tests  - запустить тесты
```

## Команды CLI

```
1       - отсортировать файл с упорядоченными элементами
2       - отсортировать файл с элементами в обратном порядке
3       - отсортировать файл с неупорядоченными элементами
4 <key> - найти значение по ключу (сначала надо выполнить команду 1-3)
5       - выход
```
