# Лабораторная работа. Вариант 3
## Галанова Екатерина Алексеевна М8О-102БВ-25

Дерево общего вида. Представление "первый потомок — следующий брат" (left-child right-sibling).

Задание по варианту: определить значение листа дерева, имеющего минимальную глубину.

## Структура проекта

- `struct.h` — описание структуры узла `Node` (поля `data`, `firstChild`, `nextSibling`).
- `tree.c` / `tree.h` — функции работы с деревом:
    - `createNode` — создание узла
    - `addChild` — добавление потомка
    - `removeSubtree` — рекурсивное удаление поддерева
    - `printTree` — вывод дерева с отступами по глубине
    - `findMinDepthLeaf` — поиск значения листа с минимальной глубиной (вариант 3)
- `cli.c` / `cli.h` — интерфейс командной строки (парсинг команд, поиск узлов, добавление/удаление).
- `main.c` — точка входа, запуск CLI.
- `tests.c` — 5 тестов через `assert`.

## Сборка

```
make all       - собрать основную программу (main.exe)
make test      - собрать тесты (test.exe)
make clean     - удалить скомпилированные файлы
```

## Запуск

```
./main.exe       - запустить программу
./test.exe       - запустить тесты
```

## Команды CLI

```
create <корень>            - создать корень
add <родитель> <значение>  - добавить узел 
delete <значение>          - удалить узел и его поддерево
print                      - вывести дерево
minleaf                    - найти лист с минимальной глубиной
exit                       - выход
```

## Тесты

В `tests.c` реализовано 5 проверок:

1. Пустое дерево — `findMinDepthLeaf` возвращает -1.
2. Один узел — лист совпадает с корнем.
3. Разветвлённое дерево — выбирается лист на минимальной глубине.
4. После удаления поддерева — результат пересчитывается.
5. После удаления корня — дерево пустое, возвращается -1.
